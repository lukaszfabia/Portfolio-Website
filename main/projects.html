Sudoku

Web game to play sudoku. Similar app to Sudoku.com. Players input is colored on blue.


Features:

  • Timer to count time of one game
  • Check button colors red numbers in the wrong positions
  • Hint button helps a player by setting a number in correct field (random choice of empty field)
  • Pause - used to stop game also blurs board
  • Remove players input

Game content:

  • 3 difficulty levels: easy, medium, hard
  • Article with tutorial
  • Article with tricks how to effectively insert numbers on a correct field

Tic tac toe

Simple logic game, implemented in Assembly mips. Player plays with computer but computer don't make random moves, only analysis board and choose best move by inserting char on the given field. Player can play 1-5 rounds. At the end of the every round the result is showed.


Priority computer moves:

  • Can win
  • Or can block a players winning move
  • Or can take mid field
  • Can take corner
  • Take whatever
